# Tic Tac Toe Game

This is a simple Tic-Tac-Toe game made using Python and Tkinter.
Two players can play turn by turn, and the game shows if someone wins or if there is a tie.

## Features
- GUI using Tkinter
- Two player mode
- Shows winner
- Highlights winning blocks
- Detects tie

## Technologies used
- Python
- Tkinter

## How to run
Make sure Python is installed, then run:
python tic_tac_toe.py
